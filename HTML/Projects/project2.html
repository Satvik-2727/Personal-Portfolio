X Percentage Calculator
Language: Python
GUI : No
Github: Repo link
Description:

This is my Percentage Calculator project made out of Python.In this calculator,number of subjects are 7. They are English, Hindi, Computer Science, Telugu,Maths,Social Science and Science. You can change the number of subjects and subject names accordingly. You would be first asked to write the total marks. Then you should write the marks of each subject. Then it would take the sum of the marks and divide it by the total marks and multiply it with 100 to find the total percentage. You can fork the repl from my github repo.
